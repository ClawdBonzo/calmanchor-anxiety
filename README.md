# CalmAnchor

**Daily anxiety recovery journal • Breathe deeper, live calmer**

A beautiful, private SwiftUI anxiety recovery companion built with 100% on-device SwiftData.

## Features

- Woofz-style onboarding flow with personalized Peace Plan generation
- Daily mood + anxiety journaling with gratitude prompts and affirmations
- Instant Panic SOS mode: 5-4-3-2-1 grounding, box breathing, and affirmation cycling
- Healing streaks with interactive calendar and 30-day structured recovery plan
- Progress charts: mood trends, anxiety levels, journaling impact, and panic event tracking
- Coping techniques resource library with step-by-step guides
- Home screen widgets for daily prompts and affirmations
- RevenueCat subscriptions with 3-day free trial highlighted
- Before/after calm transformation paywall with weekly, monthly, and annual plans
- Crisis resources and hotline quick access

## Tech Stack

- Swift 6 + SwiftUI
- SwiftData (100% local & private)
- RevenueCat
- WidgetKit + Swift Charts
- MVVM + @Observable
